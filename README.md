Password Strength Checker (CyberShield)

A modern Password Strength Analyzer built using HTML, CSS, and JavaScript

This project helps users evaluate the strength of their passwords by analyzing complexity, estimating crack time, and providing security recommendations in real time.

---

✨ Features

- 🔐 Real-time Password Analysis
- 📊 Password Strength Score (0–100)
- ⏳ Crack Time Estimation
- 🤡😎🗿 Emoji-based Security Indicator
- 📋 Password Requirement Checklist
- 🎨 Glassmorphism UI
- 📱 Responsive Design
- ⚡ Animated Strength Bar
- 🐵🙈 Fun Password Show/Hide Toggle


 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)


No installation or dependencies required.

How To Open :

1.Donwload All 3 Files In One Folder ( Eg. D:\GTU Project\Sem 5\Project )
2.Open index.html File In That Folder
3.Now You Can Enter Your Password And Check How Much Strong Your Password Is


🎯 Project Objective

The objective of this project is to spread awareness about password security by helping users create stronger passwords through instant analysis and security feedback.
